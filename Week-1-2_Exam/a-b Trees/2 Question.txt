2. Implement (a-b) trees. (Insertion and Traversing)

Input : The first line contains the a and b values separated by commas and the second line of input contains the sequence of elements to be inserted into the (a-b) trees.

Output : Print the level order traversal of (a-b) trees. If the (a-b) tree construction is not possible with the given a and b values, then print a message "(a-b) tree construction is not possible" without double quotes.
